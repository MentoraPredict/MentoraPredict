# prediction-service

**MentoraPredict — MVP Backend**

## Description

AI & Analytics service responsible for executing predictive models (logistic regression, K-means clustering), calculating failure risk scores, and providing early warning classification. Maps to RF-019.

## Architecture: Hexagonal (Ports & Adapters)

```
src/
├── domain/                    # Core ML domain rules (no framework)
│   ├── entities/              # Domain entities
│   ├── value_objects/         # Immutable value types
│   └── services/              # Domain services
├── application/               # Use-cases and orchestration
│   ├── use_cases/             # One file per use-case
│   ├── dtos/                  # Input/output schemas (Pydantic)
│   └── ports/                 # Interfaces (input & output ports)
└── infrastructure/            # Adapters
    ├── controllers/           # FastAPI routers (primary adapters)
    ├── persistence/           # PostgreSQL + MongoDB adapters
    ├── models/                # ML model files (scikit-learn)
    ├── external/              # OpenAI API client
    └── config/                # Settings and environment
```
